# LinkedIn profile audit and maintenance

## Use this reference for

- full-profile optimization reviews
- consistency checks against the context file, CV, or portfolio
- ongoing maintenance of the profile over time

## Audit workflow

1. confirm the target role or positioning before editing major sections
2. check profile architecture and structured fields first
3. review headline and About for clarity and search alignment
4. review Experience, Skills, and Featured for proof and consistency
5. review recent activity for topic alignment
6. separate hard issues from optional improvements

## Structural checklist

Check the following:

- clean public URL
- explicit location and job-preference settings when relevant
- verification status used when available
- banner supports positioning rather than being blank or generic
- headline uses standard role language
- About is first person and evidence-backed
- Experience entries contain specific scope and outcomes
- top skills are relevant and supported elsewhere on the profile
- Featured contains live proof of work, not stale filler

## Consistency checklist

The profile should align with the user's source-of-truth material on:

- job titles
- employer names
- dates
- certifications
- core technologies
- signature projects
- measurable results

If facts conflict, stop and surface the conflict explicitly.

## Activity review

Check whether recent posts and comments reinforce the same topic pillars named
in the profile. Flag these problems:

- generic praise comments
- unrelated trend chasing
- long inactivity when the user wants active creator visibility
- strong profile positioning but weak public evidence in activity

## Maintenance rules

- update sections when a real new credential, project, role, or public proof
  exists
- rotate Featured items when newer and stronger proof becomes available
- reorder skills when target positioning changes
- keep profile prose current with the user's actual role goals
- do not add speculative future achievements

## Output format for audits

When auditing a profile, organize findings into:

1. factual issues to fix first
2. discoverability improvements
3. credibility or proof-of-work improvements
4. activity recommendations

Keep recommendations actionable and tied to visible profile sections.
