# Security Policy

## Supported Versions

Security fixes are applied to the latest version on the `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for suspected security vulnerabilities.

Report security concerns privately to:

- renato.mignone@gmail.com
- home.eliainnocenti@gmail.com

Include:

- a short description of the issue
- affected files, commands, or workflows
- reproduction steps or proof of concept when available
- expected impact

We will review the report, confirm whether it is a valid security issue, and coordinate a fix before public disclosure when appropriate.
